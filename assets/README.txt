ADORNME MOBILE MENU — VERSION 2 UPDATE

HOW TO UPLOAD:
1. Go to: https://github.com/Adornmejewelry/adornme-site/blob/main/assets/mobile-menu.js
2. Click the pencil icon to edit (top right of file viewer)
3. Select all the existing code (Cmd+A) and delete
4. Open the new mobile-menu.js from this package in TextEdit or VS Code
5. Copy all contents and paste into GitHub
6. Commit message: "Fix mobile menu - half width with overlay"
7. Click "Commit changes"
8. Vercel auto-deploys in ~60 seconds
9. Test on your iPhone — the menu should now:
   - Open at 82% width with dark overlay on the right
   - Close by tapping the dark area or the X button
   - Links to Story / Journal / Gallery should work without errors
